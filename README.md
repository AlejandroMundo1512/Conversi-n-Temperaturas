# 🌡️ Conversor de Temperatura (Celsius → Kelvin & Fahrenheit)

Este proyecto es una práctica de **lógica de programación con JavaScript**.  
El programa solicita al usuario una temperatura en grados Celsius, valida que sea un número, y luego muestra la conversión a **Kelvin** y **Fahrenheit** tanto en consola como en el navegador.

---

## 🚀 Características

- Entrada de datos mediante un `input`.
- Validación de que el valor ingresado sea **numérico**.
- Conversión automática a:
  - Kelvin → `Celsius + 273.15`
  - Fahrenheit → `(Celsius × 9/5) + 32`
- Resultados mostrados en:
  - La **consola del navegador**.
  - El **DOM (pantalla del navegador)**.
- Estilos en **CSS externo**, con diseño moderno:
  - Fondo con degradado azul.
  - Caja central con sombras y bordes redondeados.
  - Botones con efectos `hover` y `active`.
  - Animación *fade-in* para mostrar los resultados.
